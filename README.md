# DNSキャッシュポイズニングテスト環境

## これは何？

アクセスしたブラウザか端末にDNSポイズニング発生する人口環境
例：tid.ac.jpにアクセスしたら別の偽サイトに飛ぶ。

## DNSキャッシュポイズニングとは？

DNSキャッシュに不正な情報を注入、偽サイトにアクセスさせる。

## 用意するもの

DNSサーバー
（権威サーバー）
偽サイト
通常サイト
